📊 Techboard – Landing Page

Landing page moderna para apresentação de uma plataforma de monitoramento em tempo real, com foco em alertas inteligentes e experiência visual impactante.

---

🚀 Sobre o Projeto

O Techboard é uma landing page desenvolvida a partir de um protótipo criado no Figma, com foco em UI moderna, hierarquia visual e conversão.

A proposta da página é apresentar uma solução de monitoramento em tempo real com alertas inteligentes, incentivando o usuário a testar a versão demo.

Este projeto reforça minhas habilidades em:

Transformar protótipos em código

Fidelidade visual ao design

Estruturação semântica

Responsividade

Organização de layout

---

🎨 Processo de Design

Antes do desenvolvimento, a interface foi planejada e estruturada no Figma, incluindo:

Definição de paleta de cores (Dark + Verde tecnológico)

Escolha tipográfica

Estruturação da hero section

Posicionamento estratégico do CTA

Simulação de mockups mobile

Essa etapa foi essencial para garantir organização visual e clareza na proposta da aplicação.
